# Revenue Proposal Copilot Plugin

Revenue Proposal Copilot is a Codex plugin for consultants, agencies, and technical service providers who need to turn messy discovery notes into priced proposals, SOW drafts, and client-ready offer packages.

## What It Does

- Converts sales notes into a clear proposal structure.
- Creates three-tier pricing packages with assumptions and exclusions.
- Flags commercial risk before a proposal is sent.
- Drafts client-facing scope, timeline, acceptance criteria, and next steps.

## Starter Prompts

- Turn these discovery notes into a priced proposal.
- Create three scope tiers for this client project.
- Find risks and missing assumptions in this SOW.

## Files

- `.codex-plugin/plugin.json`: plugin manifest.
- `skills/revenue-proposal-copilot/SKILL.md`: skill instructions.
- `scripts/offer-worksheet.md`: worksheet for collecting proposal inputs.
